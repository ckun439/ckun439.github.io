book it yourself

Tourism New Zealand is an organisation that supports NZ’s vast tourism industry and exhibits New Zealand as a tourist destination by marketing in specific groups in the global economy. The purpose of this project is to work with Tourism New Zealand and develop a single page web application. This application must allow for visitors to dynamically assess accommodation options that are given as feedback to inputs made by the user. The project that will commence will centre around Tourism New Zealand's newest campaign ‘BIY’ book it yourself campaign that focuses on users given their easy accessibility to ‘book it yourself’. The target audience of this project will focus on small groups looking to book short period holidays in New Zealand. BIY will facilitate these individuals by offering an easy, user focused, simplistic and aesthetic way of booking their next short holiday
